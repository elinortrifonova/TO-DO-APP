# TO-DO-APP
A simple todo management application that allows users to organize, filter, and track their tasks

Features
 Left Side (Uncompleted Todos)
Display all uncompleted todos
Filter todos by user ID
Sort todos alphabetically (ascending / descending)
Mark todos as completed (moves them to the right side)

 Right Side (Completed Todos)
Display completed todos
Mark todos as uncompleted (moves them back to the left side)
